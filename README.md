# thesis-autoscaling

This repository now follows a structured layout for the autoscaling thesis project.

## Structure

- app/: application entry point and container image definition
- autoscaler/: autoscaling logic, API adapters, and decision modules
- analysis/: result analysis utilities
- k8s/: Kubernetes manifests
- load/: load-generation scripts
- scripts/: helper shell scripts
- results/: experiment outputs
